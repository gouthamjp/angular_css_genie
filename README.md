## Angular CSS Genie
Angular CSS Genie is an Angular extension that automatically generates CSS classes used in your component's HTML file and adds them to the component's CSS file.

## How to Use the Extension  

1. **Update the Component's HTML File**  
   Add or modify the CSS class names in your component's HTML file as needed.  

2. **Save the Changes**  
   Simply save the HTML file (`Ctrl+S` or `Cmd+S`).  

3. **CSS Classes Added Automatically**  
   The extension will automatically generate the new class names and add them to the corresponding component's CSS file.
